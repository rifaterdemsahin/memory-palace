🪟 Windows Setup

  • Install Roblox Studio from roblox.com/create
  • Install Docker Desktop for Windows (WSL2 backend recommended)
  • Install Ollama from ollama.ai — runs as a tray app
  • Install VS Code + extensions: Luau LSP, Python, Docker, Claude Code CLI
  • Run ollama pull nomic-embed-text in PowerShell

🍎 Mac Setup

  • Install Roblox Studio from the Mac App Store or roblox.com/create
  • Install Docker Desktop for Mac (Apple Silicon or Intel)
  • Install Ollama.app — drag to Applications, it lives in the menu bar
  • Install VS Code + same extensions as Windows
  • Run ollama pull nomic-embed-text in Terminal
  • Install Claude Code CLI: npm install -g @anthropic-ai/claude-code
