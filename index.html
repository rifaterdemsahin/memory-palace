🎯 What is the Memory Palace?

A structured, living system for self-directed learning. You start with something you don't understand and walk through 7 stages until you have working code, proven results, and a shareable knowledge artefact — all anchored in memorable 3D spaces built in Roblox Studio.

AI agents (via Ollama + Qdrant) help generate, search, and embed knowledge so you can query your own palace semantically. 🧠
